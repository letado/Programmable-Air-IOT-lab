# Programmable-Air-IOT-lab
This version of the lab turns the Programmable-Air module into a serial-text field source and uses a Python gateway to expose the process state as Modbus TCP for a Node-RED HMI. The design is intended for OT-style protocol visibility, control-panel testing, and packet capture in Wireshark.
